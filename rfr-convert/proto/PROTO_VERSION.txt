Perfetto Protocol Buffer Definitions
=====================================

Source: https://github.com/google/perfetto
Branch: master
Date: 2026-02-11

These proto files were downloaded from the Perfetto repository to support
code generation for rfr-convert's Perfetto trace output.

To update these files:
1. Download the latest versions from the upstream repository
2. Update this file with the new date and commit hash
3. Run: cargo xtask gen-proto-perfetto
4. Commit the updated proto files and generated Rust code

Files vendored:
- perfetto/trace/trace.proto
- perfetto/trace/trace_packet.proto
- perfetto/trace/track_event/track_descriptor.proto
- perfetto/trace/track_event/process_descriptor.proto
- perfetto/trace/track_event/track_event.proto
- perfetto/trace/track_event/debug_annotation.proto
- perfetto/trace/track_event/thread_descriptor.proto
- perfetto/trace/track_event/counter_descriptor.proto
- perfetto/trace/track_event/chrome_*.proto (various Chrome-specific descriptors)
- perfetto/trace/track_event/log_message.proto
- perfetto/trace/track_event/task_execution.proto
- perfetto/trace/track_event/screenshot.proto
- perfetto/trace/track_event/source_location.proto
